# Project Setup & Architecture

## System Overview
The **Live Fundraising System** is a React-based web application designed to manage and display live fundraising events. It currently features a Frontend-First architecture using mock data for rapid UI prototyping.

## Tech Stack
-   **Frontend:** React (Vite)
-   **Styling:** Tailwind CSS
-   **Icons:** Lucide React
-   **Charts:** Recharts
-   **Animations:** Framer Motion
-   **Routing:** React Router DOM

## Directory Structure
```
/client
  /src
    /components  # Reusable UI components
    /pages       # Full page views
    /layout      # Layout wrappers
    /mocks       # Mock data services
    /hooks       # Custom React hooks
```

## Running the Project
1.  Navigate to `/client`
2.  Run `npm install`
3.  Run `npm run dev`

## Design System
-   **Primary Color:** Blue-600 (#2563EB)
-   **Secondary Color:** Amber-500 (#F59E0B)
-   **Fonts:** Inter (System Default San-Serif)
